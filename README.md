This is a simple project made with Pygame to replicate the iconic game 'Asteroids'. It features collision detection, game events, sprite movement, resource management, and asteroid manipulation in reaction to being shot.

# 1.

Clone the repo.

# 2.

Go to the file directory to where you cloned the repo, and set up the virtual environment:  
`uv venv`

# 3.

Run the game:  
`uv run main.py`
